package mobile.flixel

A simple button class that calls a function when clicked by the touch.

Parameters:

The x position of the button.

Y

The y position of the button.

OnClick

The function to call whenever the button is clicked.

Variables

@:value(true)allowSwiping:Bool = true

Whether you can press the button simply by releasing the touch button over it (default). If false, the input has to be pressed while hovering over the button.

read onlyjustPressed:Bool

read onlyjustReleased:Bool

label:T

The label that appears on the button. Can be any FlxSprite.

@:value([0.8, 1.0, 0.5])labelAlphas:Array<Float> = [0.8, 1.0, 0.5]

What alpha value the label should have for each status. Default is [0.8, 1.0, 0.5]. Multiplied with the button's alpha.

@:value([FlxPoint.get(), FlxPoint.get(), FlxPoint.get(0, 1)])labelOffsets:Array<FlxPoint> = [FlxPoint.get(), FlxPoint.get(), FlxPoint.get(0, 1)]

What offsets the label should have for each status.

@:value(Math.POSITIVE_INFINITY)maxInputMovement:Float = Math.POSITIVE_INFINITY

Maximum distance a pointer can move to still trigger event handlers. If it moves beyond this limit, onOut is triggered. Defaults to Math.POSITIVE_INFINITY (i.e. no limit).

@:value(false)multiTouch:Bool = false

Whether the button can use multiple fingers on it.

The properties of this button's onOut event (callback function, sound).

read onlyonOver:FlxButtonEvent

The properties of this button's onOver event (callback function, sound).

read onlyonUp:FlxButtonEvent

The properties of this button's onUp event (callback function, sound).

read onlypressed:Bool

read onlyreleased:Bool

status:Int

Shows the current state of the button, either FlxButton.NORMAL, FlxButton.HIGHLIGHT or FlxButton.PRESSED.

@:value(["normal", "highlight", "pressed"])statusAnimations:Array<String> = ["normal", "highlight", "pressed"]

What animation should be played for each status. Default is ['normal', 'highlight', 'pressed'].

Methods

destroy():Void

Called by the game state when state is changed (if this object belongs to the state)

draw():Void

Just draws the button graphic and text label to the screen.

drawDebug():Void

Helper function to draw the debug graphic for the label as well.

stampOnAtlas(atlas:FlxAtlas):Bool

Stamps button's graphic and label onto specified atlas object and loads graphic from this atlas. This method assumes that you're using whole image for button's graphic and image has no spaces between frames. And it assumes that label is a single frame sprite.

Parameters:

atlas

Atlas to stamp graphic to.

Returns:

Whether the button's graphic and label's graphic were stamped on the atlas successfully.

Called by the game loop automatically, handles touch over and click detection.
